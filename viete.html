Viète François - Az Algebrista Apa

François Viète (1540-1603) egy francia matematikus volt, aki forradalmasította az algebrát. Ő vezette be az algebrai szimbólumok használatát a polinomok és egyenletek reprezentálásához.

Fő Hozzájárulásai:

  • Az algebra szimbolikájának megalkotása
  • A polinomok és gyökeik közötti kapcsolat felfedezése
  • A trigonometria fejlesztése
  • Az egyenletek megoldásának új módszerei
  • A "Viète-féle szubsztitúció" technika

Viète munkája olyan fontos volt, hogy az "algebra apjának" nevezték.

A Viète-Tétel

Ha az x² + px + q = 0 másodfokú egyenletnek van x₁ és x₂ gyöke, akkor:

1

Gyökök Összege

x₁ + x₂ = -p

A gyökök összege egyenlő az x² együtthatójának és a lineáris tag együtthatójának hányadosa ellentettjével.

2

Gyökök Szorzata

x₁ · x₂ = q

A gyökök szorzata egyenlő a konstans tag és az x² együtthatójának hányadosával.

Általános Forma

Ha az ax² + bx + c = 0 egyenletnek van x₁ és x₂ gyöke, ahol a ≠ 0:

x₁ + x₂ = -b/a
x₁ · x₂ = c/a

Gyökök és Együtthatók Kapcsolata

A tétel közvetlenül kapcsolatot teremt a polinom gyökei és együtthatói között.

Példa: Ha x₁ = 2 és x₂ = 3, akkor x² - 5x + 6 = 0
⚖️

Szimmetria Tulajdonsága

A gyökök összege és szorzata szimmetrikus függvénye a gyökeiknek.

Megjegyzés: x₁ + x₂ = x₂ + x₁ (kommutativitás)
✓

Gyökök Helyességének Ellenőrzése

A Viète-tételt használhatjuk a gyökök helyességének ellenőrzésére.

Módszer: Helyettesítsd be és ellenőrizd az összeget és szorzatot
🔄

Fordított Alkalmazás

Ha ismerjük az x₁ + x₂ és x₁ · x₂ értékeket, felírhatjuk az egyenletet.

Képlet: x² - (x₁ + x₂)x + x₁·x₂ = 0
📊

Diszkrimináns Kapcsolat

A diszkrimináns meghatározza a valós vagy komplex gyökök számát.

Szabály: Δ > 0: két valós | Δ = 0: egy | Δ < 0: komplex
∞

Magasabb Fokú Polinomok

A Viète-tétel kiterjeszthető az n-ed fokú polinomokra.

Kiterjesztés: x³ + px² + qx + r = 0 gyökeire is alkalmazható

Gyakorlati Példa Feladatok

1. Feladat
Könnyű

Gyökök Szummájának és Szorzatának Meghatározása

Feladat: Az x² - 7x + 12 = 0 egyenlethez add meg a gyökök összegét és szorzatát!

Megoldás:

Viète-tétel alapján:

x₁ + x₂ = -(-7) = 7

x₁ · x₂ = 12

Valóban: x₁ = 3, x₂ = 4, mert 3 + 4 = 7 és 3 · 4 = 12 ✓

2. Feladat
Közepes

Egyenlet Felírása Gyökökből

Feladat: Írj fel egy másodfokú egyenletet, amelynek gyökei x₁ = 5 és x₂ = -2!

Megoldás:

Viète-tétel fordított alkalmazása:

x₁ + x₂ = 5 + (-2) = 3

x₁ · x₂ = 5 · (-2) = -10

Egyenlet: x² - 3x - 10 = 0

Ellenőrzés: x² - (x₁ + x₂)x + x₁·x₂ = x² - 3x + (-10) = 0 ✓

3. Feladat
Nehéz

Általános Forma és Szorzattényezős Alak

Feladat: A 2x² - 8x + 6 = 0 egyenlethez alkalmazd a Viète-tételt!

Megoldás:

Az általános forma: ax² + bx + c = 0, ahol a = 2, b = -8, c = 6

x₁ + x₂ = -b/a = -(-8)/2 = 8/2 = 4

x₁ · x₂ = c/a = 6/2 = 3

Szorzattényezős alak: 2(x - 1)(x - 3) = 0

Ellenőrzés: 2(x² - 4x + 3) = 2x² - 8x + 6 ✓

4. Feladat
Nehéz

Paraméter Meghatározása

Feladat: Határozd meg az m paraméter értékét, ha x₁ + x₂ = x₁ · x₂!

Megoldás:

Az x² - (m+1)x + 2m = 0 egyenlet gyökeire:

x₁ + x₂ = m + 1

x₁ · x₂ = 2m

m + 1 = 2m

m = 1

Ellenőrzés: Ha m = 1, akkor x² - 2x + 2 = 0 ✓
